# ESP32 Snake Game Controlled by MPU6050

Control a Snake game using motion gestures with ESP32, MPU6050 gyroscope, and SSD1306 OLED display.

## Features

- Motion controls
- OLED graphics
- ESP32 based
- Arduino IDE compatible

## Hardware

- ESP32
- MPU6050
- SSD1306 OLED
